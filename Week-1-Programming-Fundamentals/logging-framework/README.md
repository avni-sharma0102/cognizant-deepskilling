# Logging Framework

## What is Logging?

Logging is the process of recording application events, errors, warnings, and debugging information during program execution.

## Why Logging?

- Debugging
- Monitoring
- Error Tracking
- Maintenance

## Log Levels

- DEBUG
- INFO
- WARN
- ERROR
- FATAL

## Popular Logging Frameworks

- Log4j
- Logback
- SLF4J
- java.util.logging

## Spring Boot

By default, Spring Boot uses **SLF4J with Logback** for logging.
